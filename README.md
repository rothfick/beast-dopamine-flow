## ✨ Overview
Dopamine Flow is a playground for exploring and implementing highly engaging user interfaces. It focuses on creating seamless, intuitive, and delightful user experiences through thoughtful micro-interactions and a continuous "flow" aesthetic. This project aims to demonstrate how subtle design choices can significantly enhance user satisfaction and overall product stickiness.

## 🧱 Tech Stack
*   **TypeScript**: Provides type safety and enhances code maintainability for robust development.
*   **React**: A declarative and efficient JavaScript library for building user interfaces.
*   **Motion**: Powers fluid animations and transitions for a dynamic user experience.

## 🚀 Features
*   Responsive and adaptive UI components.
*   Example micro-interactions demonstrating immediate user feedback.
*   Smooth page transitions and state changes.
*   Modular component architecture for easy extension.

## 🛠️ Quickstart
```bash
npm install
npm run dev
```

## 🗺️ Project Structure
The project is rooted in the `src` directory, housing all application logic and components. Key configuration files like `tailwind.config.ts` and `vite.config.ts` manage styling and build processes, respectively. Tests are configured via `vitest.config.ts` and `playwright.config.ts`, ensuring robust code quality, while the `public` folder serves static assets.
